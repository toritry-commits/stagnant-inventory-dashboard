## 償却資産税クエリの変更点（会計用クエリからの差分）

### 1. 期間設定
- 月次ではなく単一期間（2025年1月2日〜2026年1月1日）に変更

### 2. 貸手リースの扱い
- 会計上は所有権移転リースとして`lease_start_at`でリース債権に振替
- **償却資産税用**: リース期間中も所有権は当社にあるため、`lease_start_at`による判定を無効化
- 除売却（`impossibled_at`）のみで減少を判定

### 3. 出力カラム
- 減価償却累計額、減損累計額、簿価をコメントアウト（償却資産税では不要）
- 取得原価のみ出力（期首・増加・減少・期末）

### 4. フィルター条件
- 取得原価 > 200,000円
- パーツ名に「棚卸対象」を含まない
- 資産分類が「賃貸用固定資産」

### 倉庫の割当ロジック

```
1. 現在の所在地（location_id → warehouse_id）
2. 履歴から最後の倉庫（stock_history → location → warehouse_id）
3. デフォルト: 船橋（warehouse_id = 21）
```

### 確認が必要な点

1. **想定との差異**: 「本社オフィス、船橋、門真だけ」を想定していたが、実際には複数の倉庫が存在
2. **法人直送・東葛西など**: 貸出中資産の最後の出荷倉庫として記録されている
3. **申告先の割当**: 各倉庫をどの市区町村に割り当てるかの確認が必要
